pypwdgen¶

This package is made to generate secure passwords.

The password are made of latin letters (both uppercase and lowercase), digits, and some punctuation marks.

pypwdgen.core._calculate_complexity(password: str) → int¶

Calculate a ‘complexity score’ based on the numbers of different character classes in the password.

Presence of the characters types a tested :

  • LOWERCASE

  • UPPERCASE

  • DIGITS

  • PUNCTUATION

For each character class, checks the presence of at least one character in the password, and store the result as a boolean.

The complexity score is the sum of these booleans (True = 1 and False = 0).

Parameters

password (str) – the password from which to calculate the complexity

Returns

the complexity score

Return type

int
